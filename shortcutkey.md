## ショートカットキー一覧

|  キー  |  操作  |
| ---- | ---- |
|  ノーツを配置する・削除する  |  1, 2, 3, 4  |
|  位置の移動	  |  ↑, ↓ |
|  ノーツの長さの変更  |  ←, →  |
|  ノーツの種類の変更  |  R  |
|  譜面を現在位置から再生  |  P  |
|  現在位置から視聴  |  L  |
|  メインメニュー  |  Esc  |
|  拡大  |  Ctrl + ↑  |
|  縮小  |  Ctrl + ↓  |
|  Triplet の オン / オフ  |  T  |
